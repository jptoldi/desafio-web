Tag: h1 até h6

Essas tags representam títulos, headers, onde o desenvolvedor vai distribuir. só um "h1" é recomendado por ser o título principal e por conta do SEO da págia, mas os subtítulos "h2" à "h6" podem ser usados à vontade entendendo sua diferenciação em tamanho. Do maior ao menor, respectivamente.

Tag: p

A tag "p" nada mais é que um parágrafo, onde será colocado blocos de texto.

Tag: mark

A tag "mark" destaca um trecho de um texto.
Texto marcado.

Tag: small

A tag "small" deixa o texto menor, usado normalmente para copyright.
Texto menor.

Tag: i

A tag "i" deixa o texto no formato itálico.
Texto itálico.

Tag: u

A tag "u" deixa o texto sublinhado.
Texto sublinhado.

Tag: strong

A tag "strong" deixa o texto em negrito.
Texto em negrito.

Tag: ol

A tag "ol" apresenta uma lista de forma numerada.

  1. Item
  2. Item

Tag: ul

A tag "ul" apresenta uma lista de forma não numerada.

  • Item
  • Item

Tag: li

A tag "li" são os itens da lista que são apresentados dentro das tags "ol" e "ul".

Tag: a

A tag "a" cria uma âncora, um hiperlink para acesso.
Link

Tag: hr

A tag "hr" cria uma linha divisória para fim de organização.

Tag: sub

A tag "sub" define uma parte do texto de forma subscrita, ou seja, com os caracteres abaixo da linha.
Texto subscrito.

Tag: sup

Ao contrário da tag anterior, a tag "sup" define a parte do texto de uma forma sobrescrita, ou seja, acima da linha.
Texto sobrescrito.

Tag: blockquote

A tag "blockquote" cria um bloco de citação ao texto marcado.

Texto citado.
